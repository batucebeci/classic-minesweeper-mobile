# Klasik Mayın Tarlası Mobil

## Amaç

Bu proje, klasik mayın tarlası oyununun mobil cihazlarda oynanabilecek bir sürümünü sunmaktadır. Kullanıcı, oyun alanındaki hücreleri açarak mayınlardan kaçınmaya çalışır.

## Çalışma Biçimi

Uygulama Expo ve React Native ile geliştirilmiştir. Oyun durumu uygulama içinde yönetilir; kullanıcı etkileşimlerine göre hücreler açılır, mayın kontrolü yapılır ve oyun sonucu belirlenir.

## Kullanım

Bağımlılıklar yüklendikten sonra proje Expo ile çalıştırılır.

```bash
npm install
npm start
```
